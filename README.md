## pboot 前后分离基础脚手架

> 基于springboot2+mybatis+spring security oauth2的前后分离脚手架,开箱即用.

### 功能包括
* 用户管理
* 用户组管理
* 组织机构管理
* 菜单管理
* 数据字典
* 操作日志
* 定时任务管理
* 菜单权限
* 数据权限（以组织机构为载体）
* spring security oauth2鉴权，可细化到按钮权限
* 更多的功能持续开发中...
